# Example Python Project

A simple Python project demonstrating basic scripts and utilities.

## Setup

```bash
pip install -r requirements.txt
```

## Scripts

- `src/hello.py` - Simple greeting script
- `src/data_processor.py` - Process CSV data
- `src/file_stats.py` - Analyze file statistics

## Usage

```bash
python src/hello.py
python src/data_processor.py
python src/file_stats.py
```
